## Как смотреть мой GitHub

1. Начните с [портфолио системного аналитика](https://github.com/AnonymousRussia007/bsa-system-analysis-portfolio).
2. Откройте карту кейсов и выберите интересующие артефакты: требования, модели данных, BPMN или интерфейсы.
3. Для дополнительной оценки технической базы посмотрите проект на C.
4. Если нужна глубина, переходите к исходным документам, таблицам и редактируемым диаграммам внутри кейсов.
