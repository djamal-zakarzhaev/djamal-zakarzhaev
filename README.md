## О профиле

Специализируюсь на системном анализе:
структурировании требований, описании бизнес-процессов,
моделировании данных, API-интеграциях
и подготовке технической документации.

В портфолио собраны учебные и практические кейсы
с аналитическими артефактами: требованиями,
моделями процессов, таблицами, диаграммами,
описаниями сценариев и интерфейсов.
